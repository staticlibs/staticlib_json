Detailed Description

template<typename Sink>
class staticlib::json::array_writer< Sink >

Writes entries to the underlying sink formatting the output as JSON array.

No more than a single entry is kept in memory in at the same time. Entries are dumped directly to underlying sink without intermediate string representation.
